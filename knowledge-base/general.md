# General Web Terms

## Terms
- **Homepage**: The main landing page of the website, reached via the root URL.
- **Header**: The area at the top of the page, containing links to major sections of the site, cart icon and personalized user menu. 
- **Footer**: The area at the very bottom of the page containing copyright info, links, and contact details.
- **Booking Widget**: A UI component used to select dates, locations, and other parameters to perform a booking search for a service and get list of a (tour, car, hotel, flight)
- **Date Range Picker**: A calendar-based component for selecting a start and end date, Located in the booking widget. On mobile it's in a modal.

## Components
- **Search Button**: Often labeled "Search Now" in the booking widget
- **Tab Element**: A clickable element in a list of tabs in the upper part of the booking widgert, used to switch between type of the search - car, flight, hotel.
- **Card**: A container for a single search result (e.g., a tour card, a car card), usually containing an image, title, price, and "View" button.

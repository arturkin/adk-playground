# Search Tour

## Metadata
- **url**: https://guidetoiceland.is
- **viewport**: desktop
- **tags**: search, tours, booking
- **priority**: medium

## Steps
1. Navigate to the homepage
2. Click on the 'Tours' tab element in the navigation
3. Click on the 'Starting date' button
3. Choose any random dates in the date range picker, few weeks in advance
4. Click on 'Search' button to see all tours

## Expected Outcome
Search results page loads with tour cards listings visible.

## Assertions
- [ ] Search results page is displayed
- [ ] At least one tour listing is visible

# Tour search flow

## Metadata

- **url**: https://guidetoiceland.is/
- **viewport**: desktop
- **tags**: search, tours, booking, filters
- **priority**: critical

## Steps

1. Navigate to the home page and click on the Tours section in the main navigation menu
2. Select 'Whale Watching' from the category dropdown menu
3. Open the date picker, navigate to the next month, and select the 12th as the starting date and the 19th as the ending date
4. Open the traveler picker, increase the count of adults and children, and set the age of the child to 5
   - [ ] Traveler configuration updated to reflect selected counts and child age
5. Click the 'Search Now' button
6. Apply filters for tour duration, specifically selecting 6 days
7. Apply location filters by selecting 'Blue Lagoon', 'Snowmobile', and 'Geysir' from the available options
8. Click on the primary image of the first tour result to open the booking page in a new window
9. Confirm the selected dates of May 12 to May 19 are correctly set on the booking page
10. Click the 'Book Now' button to proceed with the reservation

## Expected Outcome

The user is successfully directed to the booking checkout process for a 6-day tour package including Whale Watching, Blue Lagoon, Snowmobile, and Geysir.

## Assertions

- [ ] Booking page is visible
- [ ] Correct travel dates are reflected in the booking summary
- [ ] Proceed to checkout button is available

# Search Tours Filters with dates

## Metadata

- **url**: https://guidetoiceland.is/book-trips-holiday
- **viewport**: desktop
- **tags**: search, tours, filters
- **priority**: medium

## Steps

1. Locate search widget
2. In search widget, locate date picker with label "Select dates" and press on the starting date.
3. Locate a picker pop up that displays date picker
4. FILTER 1: Choose random dates, few months in advance for the 1-7 days duration.
5. Make sure from and to dates are selected correctly.
6. FILTER 2: In search widget, interact with "Choose experience" dropdown and pick "Sightseeing"
7. Perform Search
8. Locate filters on the left side of the page
9. FILTER 3: Apply any of the Duration filters
10. FILTER 4: Apply Destination filter, choose "Blue Lagoon"

## Expected Outcome

Search results page loads and displays Tour card filtered by selected criteria.

## Assertions
- [ ] Dates are set in the search widget
- [ ] Search results page is displayed
- [ ] Several Tour Cards are visible
- [ ] Requested filters are applied, check filters panel to see if they are applied
- [ ] Fail the test if one of the requested filters 1-4 was not applied
